### Random Quote App
A simple React app that displays random quotes with a "like" button to increase the like count. The like count is persistent, saved in localStorage.

## Features
- Display random quotes.

- Like a quote to increase its like count.

- Persistent like count using localStorage.

### Technologies
- React

- JavaScript (ES6)

- CSS for styling

- localStorage for data persistence

### How It Works
- Next Quote: Displays a random quote when clicked.

- Like Button: Increases the like count for the current quote. The count is saved in localStorage.
